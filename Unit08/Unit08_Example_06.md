## 1. 問題描述

### 1.1 系統說明

在一座**逆流填充吸收塔（Countercurrent Packed Absorption Tower）**中，以純水為吸收溶劑，對含有 SO₂ 的混合氣體進行洗滌吸收。氣體由塔底進入、塔頂排出；溶劑由塔頂進入、塔底排出，兩相逆向接觸，透過塔內填充物（如拉西環或鞍形填料）提供大面積氣液接觸介面，以達到高效吸收。

本案例的核心工程目標為：在給定操作條件下，計算所需的**整體傳遞單位數 $N_{OG}$** 與**填充塔高度 $H$** ，並分析液氣比（ $L/G$ ）對設計結果的影響。

**系統參數**

| 參數 | 數值 | 單位 | 說明 |
|:---|:---:|:---:|:---|
| 進塔氣體 SO₂ 莫耳分率 | $y_1 = 0.060$ | — | 塔底進氣組成（氣體進料） |
| 出塔氣體 SO₂ 莫耳分率 | $y_2 = 0.005$ | — | 塔頂出氣組成（吸收目標） |
| 進塔溶劑 SO₂ 莫耳分率 | $x_2 = 0.000$ | — | 塔頂進液為純水 |
| 整體氣相傳遞單位高度 | $H_{OG} = 0.80$ | m | 由填充物特性與系統性質決定 |

> **設計要求**：氣體中 SO₂ 吸收率達 $\eta = (y_1 - y_2)/y_1 = (0.060 - 0.005)/0.060 \approx 91.7\%$ 。

### 1.2 溶解度平衡數據

SO₂–水系統之氣液相平衡實驗數據（30°C，1 atm）如下：

| $x$ （液相 SO₂ 莫耳分率） | 0.0000 | 0.0010 | 0.0020 | 0.0030 | 0.0040 | 0.0050 |
|:---:|:---:|:---:|:---:|:---:|:---:|:---:|
| $y^*$ （平衡氣相 SO₂ 莫耳分率） | 0.000 | 0.014 | 0.030 | 0.048 | 0.068 | 0.090 |

> **注意**：平衡數據呈現輕微的上凸非線性（高濃度時偏離 Henry 定律線性關係），需以**多項式擬合**建立連續的平衡函數 $y^*(x)$ ，而非直接採用線性 Henry 定律。

## 2. 理論背景

### 2.1 氣體吸收塔的設計方程式

對於稀溶液系統（以莫耳分率表示），填充吸收塔的設計以**傳遞單位法（Transfer Unit Method）**為基礎：

$$
H = N_{OG} \times H_{OG}
$$

其中：
- $H$ ：所需填充層高度（m）
- $N_{OG}$ ：整體氣相傳遞單位數（Overall Gas-phase Transfer Units）
- $H_{OG}$ ：整體氣相傳遞單位高度（m），代表填充物質傳性能

### 2.2 整體傳遞單位數 $N_{OG}$

$N_{OG}$ 的定義積分式為：

$$
N_{OG} = \int_{y_2}^{y_1} \frac{dy}{y - y^*(x)}
$$

被積分函數 $1/(y - y^*)$ 中：
- $y$ ：氣相 SO₂ 莫耳分率（積分變數）
- $y^*(x)$ ：與液相濃度 $x$ 達到相平衡時之氣相莫耳分率（由溶解度平衡數據決定）
- $y - y^*$ ：**氣相驅動力**（Driving Force），值愈大表示傳質推動力愈強

### 2.3 操作線方程式

在稀溶液近似下，逆流吸收操作的**物料平衡操作線（Operating Line）**為：

$$
y = \frac{L}{G}(x - x_2) + y_2
$$

反解 $x$ （由 $y$ 計算對應的液相濃度）：

$$
x(y) = \frac{y - y_2}{L/G} + x_2
$$

代入積分式，得到 $N_{OG}$ 完全以 $y$ 為積分變數的表達式：

$$
N_{OG} = \int_{y_2}^{y_1} \frac{dy}{y - y^*\!\left(\dfrac{y - y_2}{L/G} + x_2\right)}
$$

### 2.4 最小液氣比 $(L/G)_{\min}$

若液氣比 $L/G$ 過低，操作線將與平衡曲線相交，導致塔底驅動力 $y_1 - y^*(x_1)$ 趨近於零，所需塔高趨於無窮大。此臨界條件定義了**最小液氣比**：

$$
\left(\frac{L}{G}\right)_{\min} = \frac{y_1 - y_2}{x_{1,eq} - x_2}
$$

其中 $x_{1,eq}$ 為使得 $y^*(x_{1,eq}) = y_1$ 的液相組成（操作線與平衡曲線的切/交點），可由求根法求解：

$$
y^*(x_{1,eq}) - y_1 = 0 \quad \Rightarrow \quad \text{由 brentq() 求解}
$$

**工程設計慣例**：取 $L/G = 1.5 \times (L/G)_{\min}$ 作為設計點。

### 2.5 `scipy.integrate.quad()` 函數說明

| 參數/說明 | 內容 |
|:---|:---|
| **函數語法** | `scipy.integrate.quad(func, a, b, args=())` |
| **主要參數** | `func`：被積分函數；`a, b`：積分下限、上限；`args`：傳遞額外參數 |
| **回傳值** | `(result, error)`：積分值與估計誤差的 tuple |
| **適用情形** | 連續可微函數之定積分、含參數插值函數積分 |
| **演算法** | 自適應 QUADPACK（Gauss–Kronrod 積分法） |

> **與梯形法的差異**：`trapezoid()` 適用於離散數據點積分；`quad()` 適用於**可連續求值的函數**，誤差可控制至機器精度（絕對誤差 $\sim 10^{-12}$ ）。

## 3. 數值計算方法與關鍵程式碼

### 3.1 計算流程

```
Step 1：平衡曲線擬合 — numpy.polyfit(x_eq, y_eq, 2) → 二次多項式係數 p
         建立函數 y_star(x) = polyval(p, x)

Step 2：求最小液氣比 — brentq 求解 y_star(x) = y1 → x_{1,eq}
         (L/G)_min = (y1 - y2) / (x_{1,eq} - x2)

Step 3：設計液氣比 — LG_design = 1.5 × LG_min
         塔底出液：x1 = (y1 - y2) / LG_design

Step 4：N_OG 積分 — quad(integrand, y2, y1)
         integrand(y) = 1 / (y - y_star(x_from_y(y, LG_design)))

Step 5：塔高計算 — H = N_OG × H_OG

Step 6：敏感度分析 — 遍歷不同 L/G 倍數，重複 Step 4–5
```

### 3.2 關鍵程式碼片段

**平衡曲線擬合**

```python
p_coeffs = np.polyfit(x_eq, y_eq, 2)          # 二次多項式擬合
def y_star(x_val):
    return np.polyval(p_coeffs, x_val)         # 建立連續平衡函數
```

**最小液氣比**

```python
x1_eq  = brentq(lambda x: y_star(x) - y1, 1e-6, 0.010)
LG_min = (y1 - y2) / (x1_eq - x2)
```

**N_OG 積分**

```python
def integrand_NOG(y_val, LG):
    x_val         = (y_val - y2) / LG + x2    # 操作線：由 y 求 x
    y_star_val    = y_star(x_val)              # 平衡函數：由 x 求 y*
    return 1.0 / (y_val - y_star_val)         # 被積分函數

N_OG, err = quad(integrand_NOG, y2, y1, args=(LG_design,))
H = N_OG * H_OG
```

## 6. 重點整理

### 6.1 關鍵計算公式

| 計算步驟 | 公式 | Python 程式碼 |
|:---------|:-----|:--------------|
| 平衡曲線擬合 | $y^*(x) = \mathrm{polyval}(\mathbf{p}, x)$ | `p = np.polyfit(x_eq, y_eq, 2)` |
| 最小液氣比 | $(L/G)_{\min} = (y_1 - y_2) / (x_{1,eq} - x_2)$ | `brentq(lambda x: y_star(x)-y1, ...)` |
| 操作線（反求 $x$ ） | $x(y) = (y - y_2)/(L/G) + x_2$ | `x = (y - y2) / LG + x2` |
| 傳遞單位數 | $N_{OG} = \int_{y_2}^{y_1} \frac{dy}{y - y^*}$ | `quad(integrand_NOG, y2, y1, args=(LG,))` |
| 塔高 | $H = N_{OG} \times H_{OG}$ | `H = N_OG * H_OG` |

### 6.2 計算結果摘要

| 項目 | 數值 | 備註 |
|:-----|-----:|:------------|
| 平衡曲線方程式 | $y^* = 1000x^2 + 13x$ | 完美擬合（誤差 = 0） |
| 最小液氣比 $(L/G)_{\min}$ | 15.2276 | 由 brentq 求解 |
| 設計液氣比 $L/G$ | 22.8413 | $= 1.5 \times 15.2276$ |
| **整體傳遞單位數 $N_{OG}$** | **4.3374** | quad 積分誤差 $< 10^{-11}$ |
| 整體傳遞單位高度 $H_{OG}$ | 0.80 m | 填充物性能參數 |
| **填充塔高度 $H$** | **3.4699 m** | $= 4.3374 \times 0.80$ |

### 6.3 分析結論

1. **多項式擬合平衡數據的重要性**：本案例平衡數據呈非線性（ $y^* = 1000x^2 + 13x$ ，有二次項），若採用線性擬合（固定 Henry 常數 $H \approx 18$ ）將導致積分結果誤差。`numpy.polyfit()` 提供簡便的多項式擬合，搭配 `polyval()` 可建立連續可微的平衡函數。

2. **`scipy.integrate.quad()` 適用場景**：本案例中被積分函數含有多項式擬合的閉合式（非原始離散數據），屬於**可連續求值的函數**，因此 `quad()` 遠優於梯形法（可達機器精度，誤差 $\sim 10^{-12}$ ）。

3. **液氣比的工程選擇**：從敏感度分析可見，接近 $(L/G)_{\min}$ （如 1.1 倍）時塔高急劇攀升至 5.92 m，而設計點（1.5 倍）為 3.47 m，增大至 3.0 倍則僅降至 2.46 m。最佳設計需在**溶劑操作成本**（正比於 $L/G$ ）與**設備投資成本**（正比於 $H$ ）之間取得平衡，工業實踐多採用 1.2–1.5 倍。
