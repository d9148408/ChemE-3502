# Unit01 電腦在化工上的應用概述 (Overview of Computer Applications in Chemical Engineering)

## 📚 單元簡介

化學工程是一門涵蓋化學反應、熱力學、質量傳遞、流體力學等多面向的複雜工程學科，現代化工問題的規模與複雜度早已超越人工計算的極限。本單元作為整個課程的起點，帶領學生全面認識電腦在化工領域中的角色與應用範疇，了解 Python 程式語言在化工科學計算中的重要性與優勢，並建立整個課程的學習路徑圖。

本單元不包含程式實作練習，而是透過豐富的概念說明、工具比較與應用案例，幫助學生在正式進入各數值計算單元前，先建立完整的「化工計算生態系統」認知框架，理解本課程各單元之間的脈絡與關聯。

---

## 🎯 學習目標

完成本單元後，您將能夠：

1. **掌握電腦在化工領域的應用全貌**：了解程序模擬、數值計算、資料分析、機器學習、CFD 等五大應用類別
2. **理解 Python 在化工計算中的優勢定位**：認識 Python 生態系統及其在科學計算中的全球地位
3. **認識化工領域常用的核心 Python 套件**：NumPy、SciPy、Pandas、Matplotlib、Scikit-learn 等
4. **了解化工領域主要商業軟體**：過程模擬軟體（Aspen Plus、HYSYS）與 CFD 軟體（ANSYS Fluent、COMSOL）
5. **建立課程學習路徑圖**：理解 Unit00–Unit18 各單元的主題、核心工具與化工應用的對應關係

## 📖 單元內容架構

### 1️⃣ 電腦在化工上的應用概述

**主要內容**：
- **化工領域的計算需求**：反應工程、熱力學、傳質傳熱、製程設計、程序控制、資料分析等典型問題的計算需求
- **五大應用類別**：
  - 程序模擬與設計 (Process Simulation & Design)
  - 數值方法求解 (Numerical Methods)
  - 資料分析與視覺化 (Data Analytics & Visualization)
  - 機器學習與人工智慧 (Machine Learning & AI)
  - 計算流體力學 (Computational Fluid Dynamics, CFD)

---

### 2️⃣ Python 程式語言在化工領域的應用

**主要內容**：
- **Python 的全球崛起**：
  - TIOBE Index 排名趨勢（2010–2025 年演變）
  - Python 超越競爭對手的四大核心優勢
  - 各行各業的 Python 採用浪潮（科技、金融、生醫、化工、製造業等）
  - 全球頂尖大學（MIT、Stanford、Harvard、台灣各大學）的 Python 必修課趨勢

- **Python 在化工計算中的優勢**：
  - 語法簡潔易學、豐富的科學套件生態、強大的資料處理能力
  - 機器學習整合、開源免費、跨平台相容、互動式開發

- **化工領域核心 Python 套件介紹**（含程式碼範例）：

  | 套件 | 功能 | 本課程應用單元 |
  |------|------|---------------|
  | **NumPy** | 高效能多維陣列、矩陣運算 | Unit03, 所有計算單元 |
  | **SciPy** | 線性代數、最佳化、ODE/PDE 求解、統計 | Unit05–Unit15 |
  | **Pandas** | 結構化資料處理、時序分析 | Unit03 |
  | **Matplotlib / Seaborn** | 資料視覺化、統計圖表 | Unit04 |
  | **Scikit-learn** | 機器學習演算法（回歸、分類、聚類） | Unit16–Unit18 |

- **Python 在化工領域的實際應用案例**：
  - CSTR 反應器穩態分析
  - 批次反應器動態模擬
  - 反應速率參數估計

- **Python 在化工領域的未來發展趨勢**：
  - 機器學習輔助材料與製程設計
  - 數位孿生 (Digital Twin)
  - 智慧型過程控制（強化學習）
  - 大數據分析平台整合
  - 物理訊息神經網路 (Physics-Informed Neural Networks, PINNs)

### 4️⃣ 化工過程模擬軟體介紹

**主要內容**：
- **過程模擬軟體的功能與定位**：穩態/動態全流程模擬、內建化學物性資料庫、熱力學模型整合
- **主要軟體比較**：

  | 軟體 | 開發商 | 適用場景 |
  |------|--------|----------|
  | **Aspen Plus** | AspenTech | 石化廠設計、能量整合 |
  | **Aspen HYSYS** | AspenTech | 天然氣處理、煉油廠 |
  | **PRO/II** | AVEVA (SimSci) | 石化、精煉 |
  | **CHEMCAD** | Chemstations | 中小型化工廠、教育培訓 |

- **Python 與過程模擬軟體的整合趨勢**：透過 COM API 以 Python 驅動 Aspen Plus 自動化批次模擬（含程式碼示例）

---

### 5️⃣ 計算流體力學 (CFD) 軟體介紹

**主要內容**：
- **CFD 在化工的重要應用**：攪拌槽、管式反應器、換熱器、填充塔、旋風分離器
- **主要 CFD 軟體介紹**：
  - **ANSYS Fluent**：有限體積法 (FVM) 核心，多紊流模型、多相流、傳熱耦合、Python Journal API
  - **COMSOL Multiphysics**：有限元素法 (FEM) 核心，多物理耦合、任意幾何自動網格剖分
- **Python 在 CFD 工作流中的角色**：自動化網格生成、邊界條件設定、後處理與視覺化
- **開源 CFD 工具鏈**：OpenFOAM、py-pde（本課程 Unit10 使用）、FEniCS/FEniCSx
